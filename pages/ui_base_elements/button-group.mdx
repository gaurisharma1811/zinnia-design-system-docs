        Buttons communicate that users have a choice among related options. Along with usage guidance, follow this content guidance:

        ### Do's 

        - Aim for one word per button. Use sentence case if needed.
        - Use parentheses to display a related number only (not related words).

        ### Don't's

        - Mismatch tenses or word types. For example if one item in the button group is a plural noun, all should be plural nouns.
