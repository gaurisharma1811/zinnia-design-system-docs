# Content

Buttons communicate action. Along with usage guidance, follow this content guidance:

  Button text must clearly explain the result of clicking a button. For accessibility, buttons contain all relevant information, even if that means re-stating a header or other visual cue on the page.
  
  Dos
  - Follow pattern: [Verb] or [Verb] [Noun]. Example: "Continue" or "Save address"
  - Sentence case with no punctuation.
