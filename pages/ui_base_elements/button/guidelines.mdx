  ## Usage

  Buttons are triggers for events or actions. They act as calls to action (CTAs). Primary buttons are used to indicate the main action of a group button set. Use concise action verbs or phrases to tell the user what will happen.

  | Only use one primary button next to other buttons. | Avoid using multiple primary buttons next to each other. This will confuse the emphasis.|
  |:---------------------------------------------------|:----------------------------------------------------|
  | Use secondary style for buttons that aren’t primary actions. | Don’t use title case capitalization or all caps. Use sentence case for buttons and button links. |
